── BYPASS METHODOLOGY ───────────────────────────────────────────────
1. ENTITLEMENT INJECTION:
   Добавить в lara.entitlements:
   - com.apple.private.security.no-sandbox (если доступен в TrustCache)
   - com.apple.security.exception.files.absolute-path.read-write
   - com.apple.security.exception.files.absolute-path.read-only
   - com.apple.private.security.storage-exempt.heritable
   -> Подписать через ldid -S<entitlements.plist> в TrustCache pipeline

2. UNENFORCED VIOLATION PATH:
   - Попытаться получить доступ к путям через unenforced vector
   - kernel sandbox логирует но НЕ блокирует (rdar://72823536)

3. VFS MAC LABEL BYPASS:
   - Использовать vfs_bypass_mac_label() из kernel exploit
   - Снять MAC label с /var/jb до mkdir()
   - Комбинировать с entitlement injection

4. TRUSTCACHE HELPER PATH:
   - create_var_jb_helper встроен в bundle → /var/tmp → posix_spawn
   - Требует: TrustCache entry + entitlements через ldid
   - Writable paths: /var/tmp, /private/var/mobile/tmp
